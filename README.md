# Jeu-du-Memory---Python
Développement d'un jeu de Memory complet en Python fonctionnant dans le terminal. Gestion de la logique de jeu (52 cartes), d'un moteur de rendu ASCII pour le tapis de jeu, et de deux modes d'interaction : un mode humain contre la machine et un mode de simulation d'IA avec différents niveaux de difficulté.
